- [x] §7 confidence **medium = "확인" 앰버 배지** → "물음표 점"으로 정정. G1 #8·ADR-0003 초기 상태와 한 몸
- [x] 제안 라벨 색 대조 — DESIGN.md 시맨틱(그린/앰버/그레이) vs G1 #8 카피(초록/호박/파랑). 근거 남기고 하나로 확정
