# 냉파 브랜드 색상 도출 보고서

## 방법
다출처 리서치 + 소크라테스식 논증. 출처: 내부 WebSearch(색채심리·오방색), **맥락 0의 독립 에이전트**(편향 배제), ui-ux-pro-max 색상 DB, taste-skill §4.2·MengTo 컬러 규율, awesome-design-md 브랜드 토큰. 가중치는 집밥 온기 ⚖ 앤티-슬롭 균형 → 이후 "먹는 즐거움" 가치 추가로 재평가.

## 핵심 긴장
냉파 정체성은 따뜻함·집밥·신뢰·**즐거움**. 그러나 taste-skill(§4.2)은 "웜 크림+브라스/클레이+에스프레소" 팔레트를 AI-slop 디폴트로 금지(기존 테라코타+크림이 여기 해당). → **뉴트럴은 쿨 클린(Apple), 온기는 액센트에만**으로 해소.

## 근거 (요약)
- **색채심리**: red/orange=식욕 자극, 쿨톤=신뢰(식욕 억제), 어스톤=자연/건강. 주황=red(식욕)+yellow(행복)=**먹는 즐거움의 색**.
- **한국 오방색**: 적/홍=火=고추장=식욕, 청=木=나물=성장·신선. 감/곶감=수확·집밥·행운.
- **ui-ux-pro-max DB(레시피앱)**: 딥 테라코타 primary + 그린 accent(단 크림 배경은 배제).
- **taste-skill §4.2**: 생크션 대안 "Terracotta+Slate", "쿨 뉴트럴 + 단일 saturated pop" → 쿨 뉴트럴 위 웜 레드/오렌지 = 합법.
- **맥락0 독립 에이전트**: 1차(식욕/한국브랜드/앤티슬롭) → 감·고추장 레드 수렴. 2차 즐거움 렌즈 2개 → **독립적으로 감/persimmon(홍시)에 수렴.**

## 소크라테스 논증 (최종)
- 正: 감·고추장 레드 유지(식욕·집밥·AA).
- 反: "즐거움" 가치엔 깊은 레드가 진지·강렬 — joy-of-food는 주황(식욕+행복).
- 合: 배달앱 오렌지·형광이 아닌 **홍시(감)** — 수확의 기쁨·친근·따뜻, redder-than-delivery로 차별화.
- 反²: 낭비방지 정체성? → **곶감** = "버릴 것 없이 인내로 더 달게" = 낭비방지 ∩ 즐거움을 한 색에.
- 反³: 밝으면 흰 텍스트 AA 붕괴 → **2단**: 브랜드/필 `#E8552D`(vivid) + 액션 `#C0391B`(AA 5.47).
- 結: **홍시(감) 퍼시먼 최종.**

- 브랜드 `#E8552D` 흰 텍스트 3.64 → **필 전용**(흰 텍스트 금지).
