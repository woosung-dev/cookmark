# 냉파(cookmark) — 확정 색상 시스템 (v1)

> 전제: **Apple식 구조**(라이트 쿨 뉴트럴 그룹 리스트 · 단일 액센트 · 최소 그림자). 온기는 **뉴트럴이 아니라 액센트에** 둔다.
> 도출: 웹 리서치 + 빈-컨텍스트 에이전트 3(A 식욕 / B 한국브랜드 / C 앤티슬롭) + ui-ux-pro-max 색상 DB + MengTo·taste-skill 규율 → 소크라테스 종합.
> 핵심 결론: **감/고추장 레드 액센트 + 쿨 뉴트럴 베이스 + 프레시 그린을 "신선/바로가능" 시맨틱으로**. (taste-skill이 금지한 "웜 크림+브라스" 디폴트를 피하면서 집밥 온기 확보.)

## 브랜드 액센트 (단일, 절제 사용 — CTA·활성·핵심 어포던스에만)
```
--accent:          #D2452E   /* 감·고추장 레드. 식욕+집밥 온기, 배달앱 오렌지(21°)보다 redder(8°)라 차별화. 흰 텍스트 버튼 AA 4.53 통과 */
--accent-pressed:  #B23A25   /* 눌림·hover. 흰 텍스트 5.96 */
--accent-text:     #B23A25   /* 흰 배경 위 '작은 액센트 텍스트'는 이걸로(AA 안전). #D2452E는 흰 위 4.53(대형/버튼용) */
--accent-tint:     #FBE7E3   /* 선택 상태·아주 옅은 배경 */
--on-accent:       #FFFFFF
```

## 뉴트럴 (Apple 쿨 클린 — 크림 금지)
```
--bg:        #F5F5F7   /* 앱 배경(iOS grouped) */
--surface:   #FFFFFF   /* 카드·시트·리스트 */
--text:      #1D1D1F   /* 본문·제목(순수 검정 아님) */
--muted:     #6E6E73   /* 보조·캡션 */
--hairline:  #E5E5EA   /* 1px 구분선 */
--sunken:    #EDEDF0   /* 입력·스켈레톤 */
```

## 시맨틱 / 제안 라벨 (색+아이콘 이중 신호)
초록은 빈-컨텍스트 B안(오방색 청=성장·나물·신선·낭비방지)의 통찰을 **보조 시맨틱**으로 흡수. 액센트 레드는 브랜드 전용.
```
/* 바로 가능 = 신선/성공 (green) */
--go-fg:   #1F6B43   --go-bg:  #E4F1E9    /* 텃밭/나물 그린. 흰 위 5.0+ */
/* 이것만 사면 가능 = 구매 유도 (amber) */
--buy-fg:  #8A5A12   --buy-bg: #FBF0DA
/* 애매하지만 가능 = 중립 (gray) */
--maybe-fg:#5B5B60   --maybe-bg:#EFEFF2
/* 부족 재료 / 에러 = 액센트 딥 레드 재사용 */
--danger:  #B23A25   --danger-bg:#FBE7E3
```

## 타이포 (Apple 규율 계승 — 세리프 디스플레이 배제)
```
--sans: 'SF Pro Display','SF Pro Text','Pretendard','IBM Plex Sans KR',system-ui,sans-serif;   /* 타이트 네거티브 트래킹(제목 -0.3~-0.5px) */
--mono: 'IBM Plex Mono',ui-monospace,monospace;   /* 매칭률·23/30·수량 tabular */
```
- 이전 냉파 커스텀의 **세리프(Nanum Myeongjo)는 뺀다** — taste-skill이 "세리프 디폴트 = 가장 많이 검증된 AI tell"로 지적, Apple 구조와도 상충. 위계는 크기·굵기·색으로.

## 그림자 (Apple 단일-그림자 철학)
```
--shadow-flat:  none;                              /* 네비바·탭바·리스트·카드 기본 */
--shadow-photo: 0 6px 20px rgba(29,29,31,.10);     /* 음식 사진이 표면 위에 놓일 때만 */
--shadow-sheet: 0 -8px 30px rgba(29,29,31,.12);    /* 바텀시트 */
```

## 접근성 요약 (검증됨)
- `#D2452E` 버튼 + 흰 텍스트 = **4.53:1 (AA 본문 통과)**. 흰 배경 위 작은 텍스트는 `#B23A25`(5.96) 사용.
- `#2E7D50/#1F6B43` 그린, `#8A5A12` 앰버-텍스트, `#5B5B60` 그레이 = 모두 흰/틴트 배경 AA 통과.
- 순수 검정(#000)·순수 흰(#fff 텍스트를 옅은 액센트 위) 금지. 액센트는 페이지 전체에서 하나로 잠금(Color Consistency Lock).

## 왜 이 조합인가 (한 줄)
쿨 클린 뉴트럴이 '신뢰·정돈'을, 감·고추장 레드 액센트가 '식욕·집밥 온기'를 맡고, 그린이 '신선·낭비방지' 시맨틱을 맡는다 — 온기와 앤티-슬롭을 동시에 만족.
